# 🎙️✨ AutoDub — AI Powered Video Dubbing Tool 🚀

> 🔥 *"Turn Any Video Into Any Language — Automatically!"*

---

## 🌟 📌 About Project (प्रोजेक्ट के बारे में)

🎯 **AutoDub** एक **AI-powered smart video dubbing tool** है जो आपके वीडियो को पूरी तरह से **automatically dub** करता है।

💡 यह टूल:

* 🎧 Audio निकालता है
* 📝 उसे Text में बदलता है
* 🌍 दूसरी भाषा में Translate करता है
* 🎙️ नई Voice Generate करता है
* 🎬 और फिर वीडियो में sync कर देता है

👉 यानी पूरा dubbing process — **Fully Automated!**

---

## 🌐 🔥 What Makes It Unique?

✨ No manual editing
✨ Fully AI-driven pipeline
✨ Multi-language support
✨ Fast & efficient processing
✨ Beginner-friendly usage

## ✨ ⚡ Features (विशेषताएं)

### 🧠 AI Intelligence

* 🎧 **Auto Transcription**
  ➤ Video audio → Text

* 🌍 **Multi-Language Translation**
  ➤ Hindi ↔ English ↔ Other Languages

* 🎙️ **Voice Generation**
  ➤ Natural AI voices

* 🎬 **Auto Syncing**
  ➤ Audio + Video perfectly merged

## 🛠️ ⚙️ Installation Guide (इंस्टॉल कैसे करें)

### 📥 Step 1: Clone Repository

```bash
git clone https://github.com/Hemant1-1/AutoDub.git
cd AutoDub
```

### 📦 Step 2: Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🚀 ▶️ Usage (कैसे उपयोग करें)

🎯 Simple Steps:

```bash
python3 main.py
```

📌 Before running:
👉 अपना video file project folder में डालें
